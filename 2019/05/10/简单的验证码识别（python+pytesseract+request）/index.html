想提高识别率嘛？(`・ω・´)
最简单的方法嘛，用PIL或者opencv库对图片进行去噪，灰度和二值化处理后，会增加识别率，尤其是那种比较难的验证码，噪点比较多，颜色也多，容易对识别结果造成影响。

其他的方法就是对这个库进行训练，可以提高识别率，不过具体的训练方法本人还在摸索。
